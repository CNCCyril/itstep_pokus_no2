git pokus 2

druhy pokus propojeni git

  1. git init --zalozeni slozky
  2. git add . --vlozeni vsech aktualnich souboru do skryte slozky
  3. git commit -m "druhy pokus vlozeni" --uvodni zprava toho co jsem vlozil, udelal
  4. git log --rekne me co se stalo, co se provedlo za operace
  5. git status --rekne co si vsechno udelal, co se prave deje
  6. git branch -M main --prejmenuje master na main
  7. git remote add origin https://github.com/CNCCyril/itstep_pokus_no2.git --nastavi kam se maji soubory nahrat na githup
  8. git push -u origin main --tento prikaz vlozi aktualni verzi vsech sobouru a slozek na githup
